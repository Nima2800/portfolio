پکیج سئوی nimalog.ir

فایل‌ها:
- index.html: نسخه اصلاح‌شده صفحه اصلی با متاهای سئو، Open Graph، Twitter Card، canonical، اسکیما Person/WebSite، اصلاح headingها، alt تصاویر و لینک داخلی پروژه‌ها.
- robots.txt: معرفی sitemap و اجازه Crawl.
- sitemap.xml: نقشه سایت شامل صفحه اصلی و صفحه‌های پروژه.
- site.webmanifest: اطلاعات پایه وب‌اپ/آیکن‌ها.
- .htaccess: تنظیمات اختیاری برای هاست Apache شامل HTTPS، دامنه canonical، cache و gzip.
- projects/: سه صفحه داخلی برای پروژه‌های مهم که به رشد روی کلمات مرتبط با نمونه‌کار کمک می‌کند.

روش نصب:
1. قبل از جایگزینی، از فایل‌های فعلی هاست بکاپ بگیر.
2. محتویات این پوشه را در روت سایت nimalog.ir آپلود کن.
3. اگر هاست Apache نیست، فایل .htaccess ممکن است اثری نداشته باشد. اگر سایت بعد از آپلود خطای 500 داد، فقط .htaccess را حذف کن.
4. در Google Search Console، آدرس https://nimalog.ir/sitemap.xml را Submit کن.
5. با URL Inspection صفحه اصلی و صفحات پروژه را Request Indexing بزن.

نکته:
مسیر CSS، JS و تصاویر فعلی دست‌نخورده مانده‌اند؛ پس باید پوشه assets فعلی روی هاست باقی بماند.
